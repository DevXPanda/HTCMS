================================================================================
5. TECH STACK
================================================================================

Backend:
  Runtime: Node (ESM)
  Framework: Express
  DB: PostgreSQL (DATABASE_URL), Sequelize ORM, raw pg pool in db.js
  Auth: JWT (jsonwebtoken), bcryptjs
  Real-time: Socket.IO (notificationSocket)
  Cron: node-cron (penalty, task generator, alert)
  Payments: Razorpay
  File: multer, pdfkit; static /uploads
  Other: cors, morgan, dotenv, express-validator, axios

Frontend:
  React 18, Vite 5, React Router 6
  UI: Tailwind CSS, lucide-react, recharts
  Forms: react-hook-form
  HTTP: axios (api.js); WebSocket: socket.io-client
  Toasts: react-hot-toast
  Date: date-fns

Env:
  Backend: PORT, DATABASE_URL
  Frontend: VITE_API_URL (default http://localhost:5000)
